# Example for stubbing/mocking unit tests

# Installation

## Clone repo

```bash
git clone https://github.com/keploy/samples-typescript && cd mocking-stubbing
yarn
```

## Start Keploy server

- See [this guide](https://github.com/keploy/keploy#quick-installation).

## Run app (tests)

- `yarn install`
- `yarn test`
